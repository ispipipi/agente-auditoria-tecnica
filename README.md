# Agente de Auditoría Técnica

Demo frontend en React + Vite + Tailwind para simular la revisión de informes técnicos, aplicar una matriz de decisión y recomendar reparación o indemnización.

## Desarrollo local

1. `npm install`
2. `npm run dev`

## Build

1. `npm run build`

## Deploy

El repositorio incluye workflow para GitHub Pages sobre la rama `main`.
